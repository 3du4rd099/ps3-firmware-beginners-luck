== How to work with the repo ==

== Linux ==

~~ This installs ps3tools to your system ~~
git clone (repo)
cd mfw/p3tools
sudo chmod +x autogen.sh
./autogen.sh
make
sudo make install

~~ This runs mfw ~~
cd mfw
sudo chmod +x ps3mfw
./ps3mfw
